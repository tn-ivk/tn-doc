# Документация дизайна TN_Doc

## Обзор

Приложение TN_Doc использует современный дизайн на основе **Material Design 3 (M3)** с единой светлой темой. Дизайн обеспечивает высокую читаемость, консистентность и современный внешний вид интерфейса.

## Цветовая палитра

### Основные цвета

| Название | CSS переменная | HEX | Описание |
|----------|----------------|-----|----------|
| Primary | `--md-primary` | `#1E88E5` | Основной синий цвет (Blue 600) |
| Primary Hover | `--md-primary-hover` | `#1565C0` | Синий для hover состояний (Blue 700) |
| Secondary | `--md-secondary` | `#43A047` | Вторичный зеленый цвет |
| Error | `--md-error` | `#E53935` | Цвет ошибок |
| Warning | `--md-warning` | `#F9A825` | Цвет предупреждений |

### Поверхности и фоны

| Название | CSS переменная | HEX | Описание |
|----------|----------------|-----|----------|
| Surface | `--md-surface` | `#FAFAFA` | Основной фон приложения |
| Surface Variant | `--md-surface-variant` | `#F1F3F4` | Фон карточек и заголовков таблиц |
| Outline | `--md-outline` | `#CFD8DC` | Цвет границ и разделителей |

### Текст

| Название | CSS переменная | HEX | Описание |
|----------|----------------|-----|----------|
| Text Primary | `--md-text` | `#212121` | Основной текст |
| Text Secondary | `--md-text-secondary` | `#5F6368` | Вторичный текст |

## Типографика

### Шрифт
- **Основной шрифт**: `'Segoe UI', 'PT Astra Sans', -apple-system, BlinkMacSystemFont, 'Helvetica Neue', Arial, sans-serif`
- **Размер базового текста**: `15px`
- **Размер текста кнопок**: `15px`
- **Вес обычного текста**: `400` (normal)
- **Вес текста кнопок**: `600` (semi-bold)

### Применение типографики
- Все текстовые элементы используют единый шрифт
- Кнопки имеют полужирное начертание для лучшей видимости
- Размер шрифта увеличен с 13.33px до 15px для улучшения читаемости

## Компоненты интерфейса

### Кнопки

**Стилизация:**
- Высота: `37px`
- Радиус скругления: `8px`
- Отступы: `6px 12px`
- Цвет фона: `--md-primary` (#1E88E5)
- Цвет текста: белый
- Hover эффект: `--md-primary-hover` (#1565C0)

**Применение:**
- Кнопки "Экспорт", "Печать", "Меню" на верхней панели
- Кнопка "Получить данные" в левой панели фильтров
- Все кнопки имеют одинаковую высоту для визуальной консистентности

### Формы и поля ввода

**Комбобоксы (select):**
- Высота: `37px` (совпадает с кнопками)
- Отступы: `6px 10px`
- Радиус скругления: `8px`
- Цвет фона: белый
- Цвет границы: `--md-outline` (#CFD8DC)

**Выбранный элемент в комбобоксе:**
- Цвет фона: `--md-primary` (#1E88E5)
- Цвет текста: белый
- Hover эффект: `--md-surface-variant` (#F1F3F4)

**Текстовые поля:**
- Высота: `37px`
- Отступы: `6px 10px`
- Радиус скругления: `8px`
- Цвет фона: белый
- Цвет границы: `--md-outline` (#CFD8DC)

### Таблицы

**Заголовки таблиц:**
- Цвет фона: `--md-surface-variant` (#F1F3F4)
- Цвет текста: `--md-text` (#212121)

**Выбранные строки:**
- Цвет фона: `--md-primary` (#1E88E5)
- Цвет текста: белый
- Применяется к строкам с классами `selected`, `active` или inline стилями

**Обычные строки:**
- Цвет фона: белый
- Цвет текста: `--md-text` (#212121)

### Карточки и панели

**Общие стили:**
- Радиус скругления: `8px`
- Цвет фона: `--md-surface` (#FAFAFA)
- Тени: легкие Material Design тени

## Структура главного окна

### Левая панель фильтров
- **Устройство**: комбобокс выбора устройства (например, "ИВК-1")
- **Документ**: комбобокс выбора типа документа (например, "Отчеты")
- **Дата начала**: текстовое поле для ввода даты
- **Дата окончания**: текстовое поле для ввода даты
- **Кнопка "Получить данные"**: основная кнопка для выполнения запроса

### Центральная область
- **Таблица результатов**: отображение найденных документов
- **Поиск**: поле поиска по документам
- **Список документов**: таблица с результатами поиска

### Правая панель
- **Настройки экспорта**: комбобоксы для выбора типа документа, протокола, формата
- **Кнопки действий**: "Экспорт", "Печать"
- **Область просмотра**: iframe для отображения PDF документов

### Верхняя панель
- **Кнопка "Экспорт"**: экспорт выбранного документа
- **Кнопка "Печать"**: печать документа
- **Кнопка "Меню"**: выпадающее меню с дополнительными функциями

### Нижняя панель (статус-бар)
- **Индикаторы состояния**: онлайн/офлайн, предупреждения
- **Цветовая схема статусов**:
  - Онлайн: зеленый (#2E7D32)
  - Офлайн: красный (#C62828)
  - Предупреждение: желтый (#F9A825)

## Принципы дизайна

### Консистентность
- Все интерактивные элементы имеют одинаковую высоту (37px)
- Единый радиус скругления (8px) для всех элементов
- Консистентная цветовая схема во всех компонентах

### Читаемость
- Увеличенный размер шрифта (15px вместо 13.33px)
- Высокий контраст между текстом и фоном
- Четкое разделение между различными областями интерфейса

### Современность
- Material Design 3 принципы
- Плавные переходы и hover эффекты
- Минималистичный и чистый дизайн

## Техническая реализация

### CSS переменные
Все цвета и размеры определены как CSS переменные в файле `material3.css`, что обеспечивает:
- Легкость изменения цветовой схемы
- Консистентность использования цветов
- Поддержку темной темы в будущем (при необходимости)

### Переопределение Bootstrap
Стили Material Design 3 переопределяют стандартные стили Bootstrap для обеспечения:
- Современного внешнего вида
- Консистентности с Material Design принципами
- Сохранения функциональности существующих компонентов

### Адаптивность
Дизайн адаптирован для различных размеров экрана с сохранением:
- Читаемости на всех устройствах
- Функциональности интерфейса
- Визуальной привлекательности

## Файлы стилей

- **`material3.css`**: основные стили Material Design 3
- **`bootstrap.min.css`**: базовые стили Bootstrap (переопределяются)
- **`site.css`**: дополнительные пользовательские стили

## Заключение

Текущий дизайн TN_Doc обеспечивает современный, консистентный и удобный интерфейс, основанный на принципах Material Design 3. Единая светлая тема с четкой цветовой схемой и типографикой создает профессиональный внешний вид приложения, сохраняя при этом высокую функциональность и удобство использования.
