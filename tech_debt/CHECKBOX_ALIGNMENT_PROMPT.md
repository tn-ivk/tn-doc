# Задача: Выровнять чекбоксы по вертикали в секции "Используемые СИ"

## Файл
`TN_Doc/Client/configurator/src/components/DeviceEditor.vue`

## Проблема
В секции "Используемые СИ" (строки 86-221) чекбоксы не выровнены по вертикали. Визуально чекбоксы в разных строках смещены относительно друг друга.

## Структура HTML (Vue template)
Секция состоит из 4 строк (`.used-si-row`), каждая содержит:
- Строка 1: "Задействовать ПР" + чекбокс (только 2 ячейки)
- Строки 2-4: "Задействовать ПП/ПВл/ПВз" + чекбокс + "Задействовать 2-ой ПП/ПВл/ПВз" + чекбокс (4 ячейки)

Классы ячеек:
- `.used-si-label` — текстовая метка
- `.used-si-control` — ячейка с чекбоксом
- `.used-si-secondary-label` — вторичная метка
- `.used-si-secondary-control` — вторичный чекбокс
- `.used-si-cell-placeholder` — пустые placeholder-ячейки

## Текущий CSS (строки ~1500-1560)
```css
.used-si-row {
  display: grid;
  grid-template-columns: 140px 32px 160px 32px;
  gap: 0.5rem;
  align-items: center;
  ...
}
```

## Требуемый результат
1. Все чекбоксы в первой колонке (ПР, ПП, ПВл, ПВз) должны быть строго на одной вертикальной линии
2. Все чекбоксы во второй колонке (2-ой ПП, 2-ой ПВл, 2-ой ПВз) должны быть строго на одной вертикальной линии
3. Чекбоксы должны быть отцентрированы в своих ячейках

## Подсказки
- Проверь, что все строки используют одинаковую grid-структуру
- Убедись, что placeholder-ячейки в первой строке занимают правильное место
- Возможно, нужно использовать `text-align: center` или `justify-self: center` для ячеек с чекбоксами
- Компонент Checkbox из PrimeVue — нужно учитывать его внутреннюю структуру через `:deep()`

## Проверка
Открой http://localhost:5000/configurator, перейди на вкладку "Устройства", выбери любое устройство и прокрути до секции "Используемые СИ". Сделай скриншот и визуально проверь выравнивание.
