# План рефакторинга `DirectoryService`

## Цель
Сделать работу со словарями стабильной, безопасной и предсказуемой (пути, кэш, наблюдение за файлами, патчи JSON).

## Проблемы сейчас
- Сбор путей с использованием текущего каталога (исправляется).
- Прямая конкатенация строк для путей JSON (устранено местами, требуется повсеместно).
- Watcher может утекать (нужна гарантированная отписка/Dispose).

## Доработки
1) Пути/безопасность
- Везде `AppContext.BaseDirectory.CombineSafe(...)`.
- Явные проверки `File.Exists`, понятные ошибки.

2) Watcher
- Ленивая инициализация; переинициализация при смене файла.
- Гарантированная отписка в `Dispose`; защита от `NullReference`.

3) Кэш JSON
- Хранить сырой JSON словарей в памяти с флагом валидности.
- Инвалидация по событию watcher и по записи.

4) Патчи
- Все патчи через `JObject`/`Replace` c проверкой ключей.
- Логировать изменения размера/контрольные суммы.

5) Тесты
- Позитив: чтение, запись, инвалидация.
- Негатив: файл отсутствует, битый JSON, отсутствие ключей.

## Этапы
- Этап 1: Пути/валидации, watcher (инициализация/очистка).
- Этап 2: Кэш и патчи, тесты.

## Критерии готовности
- Отсутствие утечек/исключений при долгих прогонах.
- Устойчивые тесты чтения/записи/инвалидации.
