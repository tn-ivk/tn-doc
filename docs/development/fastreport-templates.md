# FastReport шаблоны

## Где хранятся

- Основные шаблоны находятся в `TN_Doc/Doc/`.
- Пути к шаблонам задаются в `TN_Doc/Cfg/CfgApp.json` и/или в `TN_Doc/Cfg/Cfg*.json` через поля `PathToDocTemplateFile`.
- Пути указываются относительно корня приложения (например, `/Doc/Passport/Passport_GOSTR50.2.040(I).frx`).

## Как используется шаблон

1. Модуль документа наследуется от `DocGeneral` и формирует данные через `GetViewDoc(...)`.
2. Контроллер загружает `.frx` и передает JSON-данные в FastReport.
3. Результат экспортируется в PDF/Excel/ODS.

## Редактирование шаблона

- Используйте FastReport Designer.
- `.frx` файлы считаются бинарными — редактируйте только через дизайнер и фиксируйте изменения в репозитории.
- После правок проверьте корректность путей и сборки (см. `docs/development/building.md`).

## Частые проблемы

- **Шаблон не найден**: проверьте `PathToDocTemplateFile` в `CfgApp.json` и наличие файла в `TN_Doc/Doc/`.
- **Пустые данные в отчете**: убедитесь, что `GetViewDoc(...)` возвращает корректный JSON для шаблона.
