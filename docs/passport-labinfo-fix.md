Документ описывал проблему дублирования `LabInfo` при сохранении паспорта **в ветке с Vue‑редактором**. В текущем коде TN_Doc (v1.3.8) SPA‑редактор отсутствует, поэтому описанные шаги и ссылки на Vue‑компоненты неприменимы.

Если потребуется вернуть анализ проблемы, восстановите историю из соответствующей ветки/коммитов.
