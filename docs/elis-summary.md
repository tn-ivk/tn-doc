**Интеграция с системой истории изменений (v1.4.4+):**

Система отслеживания истории изменений полностью интегрирована с загрузкой данных из ELIS:

- **Автоматическое создание записей:** При загрузке данных из протокола ELIS автоматически создаётся запись истории с источником `DataSource.ELIS`
- **Метаданные ELIS:**
  - `ModifiedBy`: "ELIS"
  - `Comment`: "Загружено из протокола {номер протокола}" (например, "ПР-2024-12345")
  - `Source`: DataSource.ELIS
  - `ModifiedAt`: Время загрузки
- **Композабл useElisIntegration:**
  - После заполнения каждого поля вызывается `trackElisLoad(fieldKey, value, protocolNumber)`
  - История сохраняется в `documentStore.formHistory`
- **Визуальная индикация:**
  - Зелёная подсветка полей (#8fd19e) коррелирует с индикатором ELIS в истории
  - Зелёный индикатор "ЕЛИС" в правом верхнем углу поля
  - Popup с историей показывает номер протокола ELIS в комментарии
- **Приоритет источников:**
  - При последующем ручном редактировании создаётся новая запись с источником `Manual`
  - При округлении ИВК добавляется запись `IVK`
  - Индикатор всегда показывает последний источник изменений
- **Обратная совместимость:**
  - Старый флаг `ElisFilled` (boolean) автоматически пересчитывается на основе истории
  - Миграция: если `ElisFilled = true` но нет истории → создаётся запись ELIS

**Пример записи истории ELIS:**
```typescript
{
  source: 'ELIS',
  modifiedAt: '2025-01-14T10:00:00Z',
  modifiedBy: 'ELIS',
  value: '850.5',
  previousValue: null,
  comment: 'Загружено из протокола ПР-2024-12345'
}
```

- [Architecture Overview - Field History](/docs/architecture/overview.md)
